# Notebooks

This directory contains all Jupyter notebooks for the GMF Investments Portfolio Optimization project.

## Structure

| Notebook | Task | Description |
|---|---|---|
| `task1_eda.ipynb` | Task 1 | Data extraction, cleaning, EDA, stationarity testing, risk metrics |
| `task2_forecasting.ipynb` | Task 2 | ARIMA/SARIMA and LSTM model building, training, and evaluation |
| `task3_forecast_trends.ipynb` | Task 3 | Future forecasts, confidence intervals, and trend analysis |
| `task4_portfolio_optimization.ipynb` | Task 4 | Efficient Frontier, MPT optimization, portfolio recommendation |
| `task5_backtesting.ipynb` | Task 5 | Strategy backtesting vs. benchmark |

## Running Notebooks

```bash
# From the project root
jupyter lab
```

Ensure you have installed all dependencies first:
```bash
pip install -r requirements.txt
```
